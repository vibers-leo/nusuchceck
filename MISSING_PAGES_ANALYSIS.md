# 누수체크 빠진 페이지 분석 (당근마켓+토스+미소 참고)

## 📱 참고 앱 구조 분석

### 당근마켓
1. **홈 피드** - 근처 매물 목록
2. **동네생활** - 커뮤니티/후기
3. **내 근처** - 지도 기반 검색
4. **채팅** - 실시간 대화
5. **나의 당근** - 프로필/내역/설정

### 토스
1. **홈** - 잔액/계좌 요약
2. **혜택** - 쿠폰/이벤트
3. **송금** - 주요 액션
4. **전체** - 서비스 카탈로그
5. **MY** - 설정/내역

### 미소
1. **홈** - 서비스 소개
2. **서비스 신청** - 폼
3. **진행 상황** - 대시보드
4. **내 정보** - 프로필

## ❌ 빠진 페이지 (우선순위별)

### 🔥 높음 (필수)

#### 1. **서비스 소개 페이지** (`/how-it-works`)
**필요 이유:**
- 첫 방문자가 서비스 이해
- 랜딩페이지 → 가입 전 신뢰 구축
- SEO 최적화

**내용:**
- 3단계 프로세스 상세 설명 (영상 → AI 진단 → 수리)
- 실제 사용 사례 (Before/After)
- 요금 안내
- FAQ (자주 묻는 질문)

**참고:** 토스의 "서비스 소개", 미소의 "How it works"

---

#### 2. **전문가 목록/검색 페이지** (`/customers/masters`)
**필요 이유:**
- 고객이 직접 전문가 선택 가능
- 전문가 프로필/리뷰 확인
- 신뢰도 향상

**내용:**
- 지역별 전문가 목록
- 필터: 지역, 평점, 경력, 장비
- 각 전문가 카드:
  - 프로필 사진
  - 이름, 경력, 장비
  - 평균 평점 (★ 4.8)
  - 리뷰 개수
  - "상담 요청" 버튼

**참고:** 당근마켓의 "내 근처", 토스의 "전체 서비스"

---

#### 3. **리뷰/후기 페이지** (`/reviews`)
**필요 이유:**
- 사회적 증거 (Social Proof)
- 신규 고객 전환율 향상
- SEO 효과

**내용:**
- 전체 리뷰 목록 (최신순/평점순)
- 필터: 평점, 지역, 증상 유형
- 각 리뷰 카드:
  - 고객명 (익명)
  - 평점 ★★★★★
  - 증상 유형
  - 리뷰 내용
  - 작성일
  - Before/After 사진 (선택)

**참고:** 당근마켓 "동네생활", 토스 "혜택 후기"

---

#### 4. **FAQ/고객센터** (`/faq`)
**필요 이유:**
- 고객 문의 감소
- 자주 묻는 질문 해결
- 신뢰도 향상

**내용:**
- 카테고리별 FAQ:
  - 서비스 이용
  - 결제/환불
  - 보험청구
  - 전문가 매칭
- 검색 기능
- "도움이 되었나요?" 피드백
- "채팅 문의하기" 버튼

**참고:** 토스 "고객센터", 미소 "FAQ"

---

### 🟡 중간 (권장)

#### 5. **온보딩 플로우** (첫 방문자용)
**필요 이유:**
- 신규 가입자 가이드
- 이탈률 감소
- 서비스 이해도 향상

**내용:**
- 3-4페이지 슬라이드:
  1. "누수, 10분이면 해결"
  2. "영상 한 번으로 AI 진단"
  3. "검증된 전문가 매칭"
  4. "보험청구까지 자동화"
- "시작하기" / "건너뛰기" 버튼

**참고:** 토스 앱 첫 실행 시 온보딩

---

#### 6. **알림 설정 페이지** (`/customers/settings/notifications`)
**필요 이유:**
- 사용자 제어권 제공
- 스팸 방지
- UX 개선

**내용:**
- 알림 유형별 토글:
  - 견적 도착
  - 공사 시작/완료
  - 보험 승인
  - 마케팅 (선택)
- 푸시/이메일/SMS 선택

**참고:** 당근마켓/토스 "알림 설정"

---

#### 7. **결제 내역 페이지** (`/customers/payments`)
**필요 이유:**
- 투명성 제공
- 분쟁 해결
- 세금 신고 지원

**내용:**
- 전체 결제 내역 (최신순)
- 필터: 날짜, 금액, 상태
- 각 항목:
  - 날짜
  - 항목명
  - 금액
  - 상태 (완료/대기/환불)
  - 영수증 다운로드

**참고:** 토스 "전체 내역"

## 📐 디자인 가이드라인

### 당근마켓 스타일 적용
- **카드 기반 UI** - 둥근 모서리 (rounded-2xl)
- **미니멀 색상** - 흰색/회색 베이스, Primary 포인트
- **명확한 계층** - 큰 제목, 부드러운 그림자
- **여백 충분** - 답답하지 않게

### 토스 스타일 적용
- **그라디언트 강조** - CTA 버튼
- **애니메이션** - 부드러운 전환
- **간결한 텍스트** - 한 문장으로 요약
- **숫자 강조** - "10분", "95%" 등

### 미소 스타일 적용
- **단계별 진행** - 프로세스 명확히
- **안심 메시지** - 신뢰 구축
- **친근한 톤** - "~해요", "~드려요"
