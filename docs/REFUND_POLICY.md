# 누수체크 환불 규정 (초안)

> 시행일: 2026년 O월 O일
> 버전: 1.0

---

## 제1조 (목적)
이 규정은 누수체크 플랫폼을 통해 이루어진 누수 점검·수리 거래에서 발생하는 환불 기준과 절차를 정해요.

---

## 제2조 (용어)
- **고객**: 누수 점검·수리를 요청한 이용자
- **전문가**: 누수체크에 등록된 인증 전문가(마스터)
- **에스크로**: 고객이 결제한 금액을 플랫폼이 안전하게 보관하는 제도
- **3단계 결제**: 출장비 → 검사비 → 공사비 순서로 단계별 결제

---

## 제3조 (결제 단계별 환불 기준)

### 1. 출장비
| 상황 | 환불 비율 | 비고 |
|------|----------|------|
| 전문가 배정 전 취소 | **100%** | 즉시 환불 |
| 전문가 배정 후, 방문 전 취소 | **100%** | 전문가 출발 전 |
| 전문가 출발 후 고객 취소 | **0%** | 전문가 이동 비용 발생 |
| 전문가 미방문 (노쇼) | **100%** | + 전문가 패널티 부과 |
| 전문가 지각 (30분 초과) | **50%** | 고객 선택: 대기 또는 환불 |

### 2. 검사비
| 상황 | 환불 비율 | 비고 |
|------|----------|------|
| 검사 시작 전 취소 | **100%** | |
| 검사 중 고객 취소 | **0%** | 검사 장비 투입 후 |
| 누수 미확인 (정상) | **0%** | 검사 서비스 정상 제공 |
| 검사 불량 (재검사 필요) | **100%** | 분쟁 절차 통해 처리 |

### 3. 공사비
| 상황 | 환불 비율 | 비고 |
|------|----------|------|
| 공사 시작 전 취소 | **100%** | 에스크로 전액 반환 |
| 공사 중 고객 사유 취소 | **기투입 비용 제외** | 자재비 + 인건비 공제 |
| 공사 중 전문가 사유 중단 | **100%** | + 전문가 패널티 |
| 공사 완료 후 하자 발생 | **분쟁 절차** | 제5조 참조 |
| 공사 완료, 고객 미확인 (7일) | **자동 지급** | 에스크로 자동 release |

---

## 제4조 (하자보수 보증)

### 보증 기간
| 공사 유형 | 보증 기간 |
|----------|----------|
| 배관 교체 | **12개월** |
| 방수 시공 | **6개월** |
| 긴급 임시 수리 | **1개월** |
| 기타 수리 | **3개월** |

### 보증 범위
- 동일 위치, 동일 원인의 재누수에 한해 무상 재시공
- 고객 과실 (인테리어 공사 등)로 인한 재발은 제외
- 자연재해, 건물 노후화로 인한 재발은 제외

### 보증 절차
1. 고객이 하자보수 신청 (사진 필수)
2. 48시간 이내 전문가 방문 일정 조율
3. 전문가 재시공 (추가 비용 없음)
4. 전문가 미응답/거부 시 → 분쟁조정 절차 개시

---

## 제5조 (분쟁조정 절차)

### 1단계: 직접 합의 (0~48시간)
- 고객이 분쟁 접수 → 전문가에게 통보
- 전문가 48시간 이내 답변 필수
- 양측 합의 시 합의 내용대로 처리

### 2단계: 플랫폼 중재 (48시간~5영업일)
- 합의 실패 또는 전문가 미응답 시 자동 개시
- 양측 증거 제출 (사진, 영상, 견적서, 이전 대화 내역)
- 중재자(관리자)가 결정

### 3단계: 중재 결정
| 결정 유형 | 설명 |
|----------|------|
| **전액 환불** | 공사비 전액 고객 반환 |
| **부분 환불** | 하자 비율에 따라 10~90% 환불 |
| **재시공 명령** | 전문가가 무상으로 재시공 |
| **기각** | 고객 청구 기각 (전문가 정당) |

### 자동 처리 규칙
- 전문가 48시간 미응답 → **고객 유리 판정 (전액 환불)**
- 분쟁 금액 5만원 이하 → **즉시 환불 (간소화)**
- 동일 전문가 3회 이상 분쟁 → **자격 정지 검토**

---

## 제6조 (환불 처리 기간)

| 결제 수단 | 환불 소요 기간 |
|----------|---------------|
| 신용카드 | 결제 취소 후 3~7영업일 |
| 체크카드 | 결제 취소 후 3~5영업일 |
| 계좌이체 | 환불 처리 후 1~3영업일 |
| 간편결제 (토스페이 등) | 결제 취소 후 즉시~3영업일 |

---

## 제7조 (플랫폼 수수료 환불)

| 상황 | 수수료 환불 |
|------|-----------|
| 공사 시작 전 취소 | 수수료 전액 환불 |
| 분쟁으로 인한 전액 환불 | 수수료 전액 환불 |
| 분쟁으로 인한 부분 환불 | 환불 비율에 비례하여 환불 |
| 고객 단순 변심 (공사 후) | 수수료 환불 불가 |

---

## 제8조 (전문가 패널티)

| 사유 | 패널티 |
|------|--------|
| 노쇼 (미방문) | 경고 1회 + 출장비 환불 부담 |
| 48시간 분쟁 미응답 | 자동 환불 + 경고 1회 |
| 분쟁 3회 누적 (월 기준) | 신규 매칭 7일 정지 |
| 분쟁 5회 누적 (분기 기준) | 자격 정지 30일 |
| 허위 완료 처리 | 즉시 자격 정지 + 전액 환불 |

---

## 제9조 (고객 악용 방지)

| 사유 | 조치 |
|------|------|
| 허위 분쟁 접수 (증거 조작) | 분쟁 기각 + 경고 |
| 동일 건 반복 분쟁 (3회 이상) | 분쟁 접수 제한 |
| 공사 완료 후 부당 환불 요구 | 기각 + 이용 제한 검토 |

---

## 제10조 (전문가 구독료)

### 구독료
- 월 **99,000원** (부가세 포함)
- 구역 선점 + 매칭 우선권 + 플랫폼 이용료 포함

### 구독료 환불
| 상황 | 환불 |
|------|------|
| 결제 후 7일 이내, 매칭 0건 | **전액 환불** |
| 결제 후 7일 이내, 매칭 1건+ | **환불 불가** |
| 결제 후 8일 이상 | **환불 불가** (다음 달 해지 가능) |
| 플랫폼 사유 서비스 중단 | **잔여 기간 일할 환불** |
